Chatkit

public class Chatkit

This class is the entry point to the SDK.

An instance of this class will maintain a real time connection to the Chatkit service, allow access to the retrieved data, and provide methods to manipulate that data in the context of the current user.

See Initialization for details on how to get it up and running.
