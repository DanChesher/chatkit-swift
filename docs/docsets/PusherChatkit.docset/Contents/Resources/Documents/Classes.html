  • JoinedRoomsViewModel

    A view model which provides a collection of all rooms joined by the user.

    The collection of all rooms joined by the user is sorted by the view model based on the value of createdAt property of last message on the room when such message is available. When a room does not contain any messages, the view model uses the value of createdAt property of the room to determine the position of the room in the collection.
