ChatkitDelegate

public protocol ChatkitDelegate : AnyObject

A delegate protocol that describes methods that will be called by the associated Chatkit class when the status of the maintained connection to Chatkit web service have changed.

  • chatkit(_:didUpdateCurrentUser:)

    Notifies the receiver that the currently logged in user have changed.

    Declaration

    Swift

    func chatkit(_ chatkit: Chatkit, didUpdateCurrentUser currentUser: User)

    Parameters

    chatkit

    The Chatkit class that called the method.
    currentUser

    The new current user have established the connection to Chatkit web service.
