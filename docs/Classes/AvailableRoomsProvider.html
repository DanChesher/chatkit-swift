AvailableRoomsProvider

public class AvailableRoomsProvider

A provider which maintains a collection of all rooms available to the user which have been retrieved from the web service.

Properties
  • state

    The current state of the provider.

    Declaration

    Swift

    public private(set) var state: PagedProviderState
  • rooms

    The set of all rooms available to the user.

    This array contains all rooms available to the user which have been retrieved from the web service as a result of an implicit initial call made to the web service during the initialization of the class as well as all explicit calls triggered as a result of calling fetchMoreRooms(numberOfRooms:completionHandler:) method.
