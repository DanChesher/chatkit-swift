UsersProvider

public class UsersProvider

A provider which maintains a collection of users retrieved from the web service.

    The current state of the provider.

  • users

    The array of all users stored locally.

    This array contains all users retrieved from the web service as a result of an implicit initial call made to the web service during the initialization of the class as well as all explicit calls triggered as a result of calling fetchMoreUsers(numberOfUsers:completionHandler:) method.
